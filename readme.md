## LaraBlog博客管理系统

LaraBlog博客管理系统使用laravel5.1框架进行开发，系统名称中的lara取自laravel框架的前四个字母，以表达对Laravel框架的敬意。

## 系统安装环境

- PHP >= 5.5.9
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- Mysql5.6+
